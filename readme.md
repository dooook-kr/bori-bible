# 보리의 성경책 (Bori's Bible)

시력 저하로 일반 성경책을 읽기 불편하신 아버지를 위해 제작된 모바일 맞춤형 웹 성경 애플리케이션입니다. 
서버 없이 동작하는 정적 웹페이지(Static Web App)로, 직관적인 인터페이스와 가독성 향상에 초점을 맞추었습니다.

## 주요 기능
* 글자 크기 확대 및 축소 기능 (최대 40px 이상 지원)
* 기기 환경 및 사용자 선호에 따른 다크 모드 / 라이트 모드 전환
* 로컬 스토리지(Local Storage)를 활용한 원터치 책갈피 및 이어서 읽기 기능
* 한국어 성경 표준 순서 정렬

## 데이터 출처 및 기여
이 프로젝트의 핵심인 '쉽게 읽히는 성경 텍스트 데이터'는 류호현(hohyon-ryu)님의 [21cb 프로젝트](https://github.com/hohyon-ryu/21cb) 오픈소스를 활용했습니다. 
원작자가 아동과 현대인을 위해 현대어로 다듬고 요약한 마크다운 형태의 귀한 자료를, 본 프로젝트의 목적에 맞게 단일 JSON 형태로 파싱하여 적용했습니다. 텍스트 데이터를 개방해 주신 원작자에게 깊은 감사를 드립니다.

## 기술 스택
* HTML5, CSS3, Vanilla JavaScript
* Github Pages 호스팅
